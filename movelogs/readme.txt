One of the key features of Chaste Tris 1989 is that it saves and loads movelogs. A movelog is a recording of keys pressed during the game. This means that it is possible to make speedrun videos with both Chaste Tris 1989 and the original Chaste Tris,

When the game launches it loads a file named imovelog.txt and saves a file named omovelog.txt when it ends.
